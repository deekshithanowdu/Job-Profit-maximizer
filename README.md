# Job-Profit-maximizer
Job Profit Maximizer is a web app that schedules jobs with deadlines to maximize profit using a greedy algorithm. It provides real-time input analysis, interactive charts, and AI-based recommendations, displaying optimal job sequence, profit, and efficiency through a modern UI.

🚀 Job Profit Maximizer

A web-based system to solve the Job Sequencing with Deadlines problem using a greedy algorithm, enhanced with real-time analytics and AI-based recommendations.

✨ Features
📈 Maximizes profit using optimal job scheduling
⚡ Live input analysis with instant profit preview
🤖 AI-based job recommendations (missed high-profit jobs)
📊 Interactive charts and analytics dashboard
🎨 Modern UI with animations and 3D effects
🛠️ Tech Stack
Backend: Java, Spring Boot
Frontend: HTML, CSS, JavaScript
Charts: Chart.js
⚙️ How It Works
Enter jobs in format: JobID, Deadline, Profit
System schedules jobs using greedy algorithm
Displays:
Optimal job sequence
Total profit
Efficiency & missed jobs
AI suggests better job choices
▶️ Run the Project
Backend
mvn spring-boot:run
Frontend

Open index.html in browser

🌐 API
POST /api/schedule
